# Code of Conduct

x-harness uses a lightweight contributor covenant for a small open-source CLI project.

## Our standards

Participants are expected to:

- Be respectful, constructive, and patient.
- Critique ideas and code, not people.
- Assume good intent while still asking for evidence.
- Avoid harassment, discriminatory language, personal attacks, and spam.

## Scope

This applies to issues, pull requests, discussions, code review, and project-related communication.

## Enforcement

Maintainers may remove comments, close issues, block users, or restrict participation when behavior is abusive or persistently disruptive.

To report a conduct concern, contact the repository maintainer through GitHub.
