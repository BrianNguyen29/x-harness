# Preview Examples

These examples illustrate advanced or roadmap features. They are not required for v0.1 and do not redefine the default lightweight flow.

## Contents

- `02-assisted-agent/feature-intake.md` + `story.md`: Feature intake and story packet (v0.2 preview).
- `04-verify-fail-recovery/`: Recovery flow after a failed verify (v0.3 preview).
- `05-product-intake-to-accepted-completion/`: Full product intake flow (v0.2+ preview).

## Note

Preview files may reference schemas, templates, or commands that are not yet stable.
