# Support

Use GitHub Issues for reproducible bugs, documentation gaps, and feature
requests.

For security issues, do not open a public issue. Follow [SECURITY.md](SECURITY.md)
so the report can be handled privately.

For usage questions, include:

- x-harness version or commit.
- Node.js and npm versions (or Go version if using the native CLI).
- The command you ran.
- A minimal completion card or fixture that reproduces the behavior.
